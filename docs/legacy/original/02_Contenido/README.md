# Contenido - Preguntas y Datos

Documentación sobre cómo crear y estructurar preguntas.

## Archivos en esta carpeta

- **ESTRUCTURA_DATOS_PREGUNTAS.md** - Especificación completa del formato JSON
- **CONTENIDO_PREGUNTAS_GUIA.md** - Guía paso a paso para crear preguntas
- **GUIA_CONTENIDO_PREGUNTAS.md** - Guía alternativa más detallada
- **SISTEMA_CONTENIDO_AVANZADO.md** - 10 tipos de contenido disponibles

## Cuándo consultar esta carpeta

- Vas a agregar nuevas preguntas al banco
- Necesitas entender el formato de datos
- Quieres ver ejemplos de contenido complejo
- Necesitas validar una pregunta

## Flujo de lectura recomendado

1. `CONTENIDO_PREGUNTAS_GUIA.md` - Comienza aquí
2. `ESTRUCTURA_DATOS_PREGUNTAS.md` - Entiende el formato
3. `SISTEMA_CONTENIDO_AVANZADO.md` - Tipos de contenido
4. Ve a `06_Ejemplos/EJEMPLOS_PREGUNTAS_AVANZADAS.js` - Ve código real
