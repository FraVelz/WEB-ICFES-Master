# Diseño - Estilos y UI

Documentación sobre diseño visual, estilos y mejoras de interfaz.

## Archivos en esta carpeta

- **DESIGN_IMPROVEMENTS.md** - Mejoras de diseño implementadas
- **STYLES_GUIDE.md** - Paleta de colores, tipografía, espaciado
- **MEJORAS_IMPLEMENTADAS_DICIEMBRE.md** - Cambios recientes

## Cuándo consultar esta carpeta

- Vas a crear nuevos componentes
- Necesitas mantener consistencia visual
- Necesitas saber qué colores usar
- Quieres ver las mejoras recientes

## Flujo de lectura recomendado

1. `STYLES_GUIDE.md` - Entiende el sistema de diseño
2. `DESIGN_IMPROVEMENTS.md` - Ve cambios implementados
3. `MEJORAS_IMPLEMENTADAS_DICIEMBRE.md` - Novedades
