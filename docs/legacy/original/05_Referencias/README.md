# Referencias - Guías Generales

- **GUIA_DOCUMENTACION.md** - Índice con matriz de decisión de documentación
- **RESUMEN.md** - Resumen ejecutivo del proyecto completo

- No sabes por dónde empezar

- Buscas una guía de referencia

## Flujo de lectura recomendado

1. `GUIA_DOCUMENTACION.md` - Matriz de decisión
2. `RESUMEN.md` - Overview del proyecto
3. `VISUALIZACION_SISTEMA.md` - Diagramas
