# Arquitectura - Estructura del Proyecto

Documentación sobre cómo está organizado el código fuente.

## Archivos en esta carpeta

- **ESTRUCTURA_README.md** - Resumen ejecutivo de la arquitectura
- **ESTRUCTURA_ARCHIVOS.md** - Tabla completa de ubicación de componentes
- **ESTRUCTURA_VISUAL.md** - Visualizaciones ASCII de carpetas
- **ESTRUCTURA_NUEVA.md** - Detalles de Feature-Based Architecture
- **ESTRUCTURA_COMPLETA.md** - Guía completa de todas las carpetas
- **ARQUITECTURA_SERVICIOS.md** - Servicios y Firebase
- **AUDITORIA_PAGINAS.md** - Análisis de páginas redundantes

## Cuándo consultar esta carpeta

- Buscas dónde está un componente específico
- Quieres agregar una nueva página
- Necesitas entender por qué la estructura es así
- Quieres agregar un nuevo feature

## Flujo de lectura recomendado

1. `ESTRUCTURA_README.md` - Overview
2. `ESTRUCTURA_ARCHIVOS.md` - Encuenttra componentes
3. `ESTRUCTURA_VISUAL.md` - Ve diagramas
4. `ESTRUCTURA_NUEVA.md` - Entiende detalles técnicos
