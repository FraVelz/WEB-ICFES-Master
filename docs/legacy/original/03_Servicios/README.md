# Servicios - Firebase y Lógica de Negocio

Documentación sobre servicios, Firebase y arquitectura técnica.

## Archivos en esta carpeta

- **CHECKLIST_FIREBASE.md** - Configuración y verificación de Firebase
- **DOCUMENTACION.md** - Documentación general del sistema

## Cuándo consultar esta carpeta

- Necesitas configurar o verificar Firebase
- Quieres entender cómo funciona el backend
- Necesitas agregar un nuevo servicio
- Necesitas acceso a documentación de APIs

## Flujo de lectura recomendado

1. `CHECKLIST_FIREBASE.md` - Verifica la configuración
2. `DOCUMENTACION.md` - Entiende el sistema
3. Ve a `01_Arquitectura/ARQUITECTURA_SERVICIOS.md` - Detalles técnicos
